kvkbd
=====

Virtual Keyboard for KDE

## Requirements
* Plasma 5 & Qt 5.9+ Gui Core Widgets Xml X11Extras DBus
* KF5 5.40+ I18n Config CoreAddons IconThemes XmlGui WindowSystem DocTools

## Compile and install
```cmake
git clone https://github.com/bvbfan/kvkbd
cd kvkbd
mkdir build
cd build
cmake ..
make
sudo make install
```
